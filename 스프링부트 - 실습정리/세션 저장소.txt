세션 저장소

1. 톰캣 세션 사용
	- 디폴트 값. 
	- 톰캣에 세션이 저장되기 때문에 2대 이상의 WAS가 구동되는 환경에서는 톰캣간 세션 공유를 위한 추가 설정 필요	
2. MySQL과 같은 DB 사용
	- 여러 WAS간 공용 세션 사용할 수 있는 가장 쉬운 방법.
	- 로그인 요청마다 DB IO발생하여 성능상 이유가 발생될 수 있음.
	- 보통 로그인 요청이 많이 없는 백오피스, 사내 시스템 용도
3. Redis, Memcached와 같은 메모리DB 사용
	- B2C에서 가장 많이 사용하는 방식.
 	- 실제 서비스하기 위해서는 Embedded Redis와 같은 방식이 아닌 외부 메모리 서버가 필요.
